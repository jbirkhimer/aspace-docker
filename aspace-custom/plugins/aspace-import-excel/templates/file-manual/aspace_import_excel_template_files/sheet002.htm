JUST NOTES FOR NOW...
Creating new Resource records via spreadsheet import is out of scope.
Column D will be blank if adding Archival Objects to a Resource that does not have any.
Use Hierarchical Relationship=1 for the top level of Archival Objects to be added to a Resource that does not have any.
Ref_ids will be generated for Archival Objects and Digital Objects created with this process.
Date Era and Calendar will be set to blank during import.
barcode will not be added to an existing top container. 
Locations must be created and/or linked to containers in the ArchivesSpace UI. They cannot be created or linked via spreadsheet import.
